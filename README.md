# Klasifikasi-dengan-Decision-Tree-dan-Random-Forest

## Deskripsi

Proyek ini merupakan tugas praktik klasifikasi menggunakan algoritma Decision Tree dan Random Forest. Dataset yang digunakan adalah dataset buatan sendiri dengan tema film, yang bertujuan untuk memprediksi apakah sebuah film tergolong sukses atau tidak.


## Dataset

Dataset berisi beberapa fitur, yaitu:

* Budget → biaya produksi film
* Duration → durasi film
* Genre → jenis film (Action, Comedy, Drama)
* Rating → penilaian film
* Success → target (1 = sukses, 0 = tidak sukses)


## Tahapan Pengerjaan

1. Membuat dataset film
2. Preprocessing data (encoding fitur kategorikal)
3. Membagi data menjadi data latih dan data uji
4. Membangun model:

   * Decision Tree
   * Random Forest
5. Melakukan prediksi
6. Evaluasi model menggunakan:

   * Accuracy
   * Confusion Matrix
   * Classification Report
7. Membandingkan hasil kedua model
8. Visualisasi Decision Tree
9. Analisis pengaruh parameter n_estimators pada Random Forest



## Hasil

Hasil menunjukkan bahwa kedua model memiliki performa yang sangat baik pada dataset yang digunakan. Random Forest cenderung lebih stabil, sedangkan Decision Tree lebih mudah dipahami.
